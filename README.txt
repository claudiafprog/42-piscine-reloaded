# 42 Piscine Reloaded

This repository contains the exercises completed during the 42 Piscine Reloaded.

## Overview

The Piscine Reloaded is a set of exercises designed to reinforce and consolidate the core concepts learned during the 42 Piscine.

It focuses on strengthening understanding of C programming fundamentals, logic building, and structured problem-solving.

## Structure

Each exercise is organized in its own folder (exXX), following a progressive increase in difficulty and conceptual depth.

## Goals

- Reinforce core C programming concepts
- Improve logical thinking and problem-solving skills
- Practice writing clean, structured, and readable code
- Develop consistency and autonomy in development

## Notes

These exercises were completed as part of the 42 learning methodology, where emphasis is placed on understanding, discipline, and incremental progression.

The repository reflects a learning process rather than production-ready code.
